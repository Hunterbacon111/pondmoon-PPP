四步搭建全流程

1

注册 Claude & 安装 Claude Code

  • 前往 claude.ai 注册账号，选择 Monthly Plan
  • Pro 计划 $17/月 —— 日常够用
  • Max 计划 $200/月 —— 重度使用者
  • 下载 Claude Code（桌面版 或 Terminal 版均可）
  • 本地建一个项目文件夹，放哪儿都行，空间够、能找到就行
2

用 Claude Code 造你想要的工具

Claude Code 的核心：你说需求，它写代码。几乎能造出任何定制工具：

  • T-12 自动分析器 —— 上传物业 T-12 报表，自动分析和对比
  • 邮件语气模拟器 —— 学习你的语气，帮你生成专业邮件
  • Rent Roll 对比工具 —— 高亮不同时期的变化
  • 贷款计算器 —— 自定义参数，敏感度分析
  • 暂时没灵感？没关系。记住它能造一切，等你有需求随时回来
3

搭建 OpenClaw AI 助手

这是最"工程"的一步，但用 Claude Code 帮你装会简单很多。详见下方的安装流程。

4

调教 & 开始使用

  • 在 OpenClaw Dashboard 中给它取名、设置性格
  • 连接 WhatsApp / Telegram / 飞书等聊天软件
  • 开始布置任务：查天气、查航班、订酒店、安排日程...
  • 在使用过程中教它学习新 Skills，越用越聪明

为什么要用 Docker（保护区）？
你的鱼（OpenClaw）是条斗鱼，非常敏感、容易过激。所以在鱼缸（服务器）里又加了一个保护区（Docker），让斗鱼安全地适应生活。如果它搞出什么问题，在保护区里就能修改，不会污染鱼缸里的其他生态。

OpenClaw 安装流程（通过 Claude Code）

1

选购服务器（鱼缸）

推荐 DigitalOcean（~$10-12/月，纽约节点），内存 2GB 以上。
也可选阿里云、腾讯云海外节点，更便宜。

2

获取 SSH 连接

拿到服务器的 SSH 信息，例如 ssh root@你的IP

3

Claude Code 连接服务器

在本地 Claude Code 里建立 SSH 连接，把本地和远程打通

4

在 Docker 里安装 OpenClaw

告诉 Claude Code："在 Docker 里安装 OpenClaw"。它会帮你处理所有配置细节。
OpenClaw 官网有一行代码完成下载，非常顺滑。

5

配置大模型 API

不差钱 → Claude API  |  省钱 → Kimi API

6

开通网关（交给 Claude Code）

网关配置比较花时间，让 Claude Code 帮你开，出问题它也能帮你排查。

常见问题

我完全不懂代码，能搞吗？
能。Claude Code 本身就是帮你写代码的，你只需要用自然语言描述需求。安装服务器和 OpenClaw 的过程中，遇到问题都可以问 Claude Code。
为什么不直接用 ChatGPT / Claude 聊天，还要搞 OpenClaw？
ChatGPT 和 Claude 是"被动"的——你问它才答。OpenClaw 是"主动"的——你可以给它布置定时任务、连接各种服务、让它自主执行工作，更像一个真正的助手。
Kimi 2.5 是什么？
Kimi 是月之暗面（Moonshot AI）出品的大模型，性能不错，API 价格比 Claude 便宜很多。作为 OpenClaw 的"大脑"来用，性价比很高。
服务器挂了怎么办？
因为用了 Docker，你的 OpenClaw 配置都在容器里。服务器重启后，Docker 容器会自动恢复。实在出问题，用 Claude Code 重新连上去修就行。
我在国内，怎么办？
两个办法：

方法一：科学上网。配置好网络工具后，上述所有流程都可以正常进行，Claude Code、DigitalOcean、OpenClaw 都能正常访问。

方法二：用 Kimi Claw（推荐国内用户）。Kimi 官方推出了一键部署 OpenClaw 的服务，不需要自己买服务器、装 Docker，直接在 Kimi 平台上就能搞定，省去了所有运维步骤。前往 kimi.com/bot 了解详情。
